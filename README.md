# StudyStreak

macOS 向けの学習時間トラッカーです。  
カテゴリを作って学習セッションを記録し、ダッシュボードとウィジェットで振り返れます。

## 主な機能

- 学習セッションの開始 / 終了
- カテゴリ管理（追加・削除、色ラベル付き）
- セッション履歴の確認と削除
- ダッシュボード表示
  - 今日の学習時間
  - 今週の学習時間
  - 連続学習日数（1秒でも学習した日は達成）
  - 最近のセッション
- ウィジェット（`small` / `medium`）
  - `small`: Current Streak 中心
  - `medium`: Current Streak + Today / Total
